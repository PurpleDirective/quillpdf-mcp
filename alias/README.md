# quillpdf

This is the official npm name-claim for **[QuillPDF](https://quillpdf.com)** — free, local-first PDF tools. It re-exports the real package, **[quillpdf-mcp](https://www.npmjs.com/package/quillpdf-mcp)**: local PDF operations (merge, split, rotate, watermark, Bates-number, clean metadata, page-count) as an MCP server for AI agents and a CLI. Your files never leave your machine.

```bash
npx quillpdf --help        # the CLI (via this alias)
npx -p quillpdf-mcp quillpdf --help   # the real package directly
```

Source, docs, and issues: <https://github.com/PurpleDirective/quillpdf-mcp> · MIT.
